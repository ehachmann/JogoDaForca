![](https://imgur.com/wl7U8zF.gif)

# Introdução 

Nosso desafio é desenvolver um jogo de forca em console. O computador escolherá (de maneira aleatória) uma palavra entre várias possibilidades e o jogador deve chutar, letra por letra, até adivinhá-la.

O jogador poderá cometer 5 erros antes de perder.

# Passo a passo 

Definir estrutura básica do programa e receber input do usuário
Exibir uma dica da palavra oculta escolhida
Verificar a validade de uma letra digitada pelo usuário e contabilizar erros
Exibir um desenho da forca que é atualizado a cada tentativa
Gerar uma palavra aleatória para ser adivinhada
Desenho da Forca
